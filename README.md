### Number.py
| arguments  | description  |
|:-:|:-:|
|   | Add 0 into filenames, only files named with number  |
| -r  | Replace the current filename to a number |
| -rf  | Replace the current folder name to a number |

##### <-rf>
| before | after |
|:-:|:-:|
| /git-folder | /0 |
